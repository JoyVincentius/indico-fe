# User Management App

A small React + Material‑UI demo that showcases:

- **Fetching** users with pagination & full‑text search
- **Adding** a new user
- **Deleting** an existing user (with confirmation dialog)

## Prerequisites

- Node.js (>= 16)
- npm (>= 8)

## Installation

```bash
git clone <repo-url>
cd user-management-app
npm install
